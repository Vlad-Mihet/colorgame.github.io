What is this game about and what should you do?

The game consists of a rgb code for a color being displayed at the top of screen

The objective of the game is to choose, from 3, 6, or 9 color blocks the color block that represents the rgb code present at the top of the screen.

The difficulty may be chosen from the menu below the rgb code. There are 3 options; Easy, Medium and Hard.

The RGB is composed of acronyms that stand for Red, Green and Blue. The values for these 3 colors range from 0 all the way to 255 each.
